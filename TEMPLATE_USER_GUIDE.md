# 範本管理與匯入匯出 - 使用指南

## 🎯 功能簡介

手動配對系統現在支援**範本管理**和**匯入匯出**功能，讓您可以：
- 💾 儲存常用的配對模式
- 📂 快速載入先前的配對
- 📤 匯出配對備份
- 📥 匯入他人分享的配對

---

## 💾 儲存範本

### 何時使用
- 完成了一套滿意的配對設定
- 想要保存這個配對模式供以後使用
- 需要建立多個配對方案進行比較

### 操作步驟
1. 在手動配對頁面完成部分或全部配對
2. 點擊頂部的「💾 儲存範本」按鈕
3. 在彈出視窗中輸入範本名稱
   - 例如：「標準配對」、「女雙優先」、「年輕組合」
4. 點擊「確定儲存」
5. 看到「範本已儲存」訊息即完成

### 💡 小技巧
- 使用有意義的名稱，方便日後識別
- 可以儲存多個不同的範本
- 範本會記錄當時的賽事設定（人數/點數/輪數）

---

## 📂 載入範本

### 何時使用
- 想要套用先前儲存的配對模式
- 需要快速開始新一輪賽事
- 想要比較不同的配對方案

### 操作步驟
1. 點擊「📂 載入範本」按鈕
2. 查看範本列表，每個範本顯示：
   - 📝 範本名稱
   - 📅 儲存日期和時間
   - ⚙️ 賽事設定（X人/Y點/Z輪）
3. 點擊想要的範本右側的「載入」按鈕
4. 確認載入（會覆蓋目前配對）
5. 範本配對會自動填入

### ⚠️ 注意事項
- 載入會**覆蓋目前的配對**，請先確認
- 如果範本設定與目前設定不同，會顯示警告
- 如果找不到對應的選手，該位置會顯示空白
- 空白位置需要手動重新選擇選手

---

## 🗑️ 刪除範本

### 操作步驟
1. 點擊「📂 載入範本」
2. 找到要刪除的範本
3. 點擊右側的「刪除」按鈕
4. 確認刪除
5. 範本會從列表中移除

---

## 📤 匯出設定

### 何時使用
- 需要備份配對設定
- 想要分享配對給其他管理員
- 要在不同電腦或瀏覽器使用同一配對
- 擔心瀏覽器資料遺失

### 操作步驟
1. 完成配對設定
2. 點擊「📤 匯出設定」按鈕
3. 系統自動下載 JSON 檔案
4. 檔案會儲存到您的下載資料夾
5. 檔名格式：`配對設定_2025-12-29.json`

### 檔案內容
匯出的 JSON 檔案包含：
- ✅ 完整的配對資訊
- ✅ 選手詳細資料（ID、姓名、年齡、性別）
- ✅ 賽事設定資訊
- ✅ 儲存時間戳記

### 💡 建議用途
- **備份**：定期匯出重要配對
- **分享**：透過電子郵件或雲端分享檔案
- **版本控制**：匯出不同版本的配對方案
- **長期保存**：不受瀏覽器限制

---

## 📥 匯入設定

### 何時使用
- 要載入先前匯出的備份
- 收到其他人分享的配對檔案
- 要在新電腦或瀏覽器使用配對
- 要恢復不小心修改的配對

### 操作步驟
1. 點擊「📥 匯入設定」按鈕
2. 選擇 JSON 檔案（先前匯出的檔案）
3. 系統自動讀取並解析檔案
4. 確認匯入（會覆蓋目前配對）
5. 配對會自動填入

### 🔍 匹配邏輯
系統會智能匹配選手：
1. **優先**：根據選手 ID 匹配
2. **其次**：根據選手姓名匹配
3. **找不到**：該位置顯示空白

### ⚠️ 注意事項
- 匯入會**覆蓋目前的配對**
- 選手陣容必須相同或相似才能正確匹配
- 建議在新增完選手後再匯入
- 匯入後檢查是否有空白位置需要補充

---

## 📋 實際應用場景

### 場景 1：年度賽事
**情況：** 每年舉辦相同規則的賽事

**做法：**
1. 第一年完成配對後儲存為「2025標準配對」
2. 次年新增選手名單
3. 載入「2025標準配對」範本
4. 系統自動套用去年的配對模式
5. 微調新加入的選手配對

**好處：** 省時、一致、可靠

---

### 場景 2：AB 方案比較
**情況：** 想測試不同配對策略

**做法：**
1. 設定方案 A 並儲存為範本
2. 修改為方案 B 配對
3. 需要時載入方案 A 範本比較
4. 選擇最佳方案開始賽事

**好處：** 靈活、可逆、易比較

---

### 場景 3：多地賽事
**情況：** 在不同城市舉辦相同賽事

**做法：**
1. 總部完成標準配對
2. 匯出為 JSON 檔案
3. 傳送給各地管理員
4. 各地匯入後調整選手

**好處：** 統一、高效、易管理

---

### 場景 4：安全備份
**情況：** 擔心資料遺失

**做法：**
1. 完成配對後立即匯出
2. 儲存到雲端硬碟
3. 需要時隨時匯入恢復
4. 定期更新備份檔案

**好處：** 安全、跨裝置、持久

## ❓ 常見問題

### Q1: 範本和匯出有什麼不同？
**A:** 
- **範本**：儲存在瀏覽器內，快速載入，適合常用配對
- **匯出**：儲存為檔案，可備份和分享，適合長期保存

建議兩者都使用：常用的存範本，重要的匯出備份。

---

### Q2: 載入範本後選手顯示空白怎麼辦？
**A:** 可能原因：
- 該選手的 ID 或姓名已變更
- 選手已被刪除
- 選手尚未新增

**解決方法：** 手動從下拉選單重新選擇選手

---

### Q3: 可以跨不同設定載入範本嗎？
**A:** 可以，但不建議：
- 系統會顯示警告訊息
- 點數不同可能導致部分配對無效
- 人數不同可能找不到對應選手

建議只在相同設定下載入範本。

---

### Q4: 範本會佔用多少儲存空間？
**A:** 
- 每個範本約 5-10 KB
- 瀏覽器 localStorage 通常有 5-10 MB
- 可儲存數百個範本
- 不用擔心空間問題

---

### Q5: 匯出的檔案可以用 Excel 開啟嗎？
**A:** 
- 檔案格式是 JSON，不建議用 Excel
- 可用文字編輯器查看
- 主要用途是系統間傳輸
- 未來可能支援 Excel 格式

---

### Q6: 刪除範本可以復原嗎？
**A:** 
- 無法直接復原
- 如果有匯出備份可以匯入
- 刪除前會要求確認
- 重要範本建議先匯出備份

## 🎯 最佳實踐

### ✅ 建議做法
1. **命名規範**：使用描述性名稱「2025春季賽_女雙優先」
2. **定期備份**：完成配對後立即匯出
3. **多版本管理**：儲存多個方案供比較
4. **雲端儲存**：將匯出檔案存到 Google Drive 或 OneDrive
5. **測試驗證**：載入範本後檢查所有配對

### ❌ 避免做法
1. 不要使用含糊的範本名稱「測試1」「暫存」
2. 不要在不同設定下強制載入範本
3. 不要刪除還在使用的範本
4. 不要忘記定期清理過期範本
5. 不要依賴單一備份方式

## 🚀 進階技巧

### 技巧 1：快速複製
想要快速複製配對到另一個賽事：
1. 匯出當前配對
2. 開啟新賽事
3. 匯入剛才的檔案
4. 微調即可使用

### 技巧 2：範本版本控制
管理多個版本的配對：
1. 儲存「v1.0標準配對」
2. 修改後儲存「v2.0改良版」
3. 繼續改進儲存「v3.0最終版」
4. 可回頭載入任何版本

### 技巧 3：團隊協作
多人協作管理賽事：
1. 主管理員完成配對
2. 匯出檔案分享給助理
3. 助理匯入後可查看和學習
4. 提出建議後由主管修改
